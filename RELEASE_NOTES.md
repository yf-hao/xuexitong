# 版本说明 (Release Notes) - V0.1

## 📅 更新日期: 2026-01-24

## ✨ 新增功能 (New Features)

### 1. 班级管理系统 (Class Management)
- **批量导入学生**: 支持解析树维教务系统导出的 Excel 成绩登记册，将学生批量添加至指定班级。
- **班级维护**: 新增重命名班级、归档（删除）班级功能。
- **教师分配**: 支持将班级分配给教学团队中的特定教师。
- **新建班级**: 提供了新建班级的入口（API已集成）。

### 2. 成绩权重管理 (Grade Weights)
- **可视化配置**: 提供了直观的百分比滑块/输入框，用于调整各项成绩权重（如视频、测验、作业、考试等）。
- **一键同步**: 支持将当前权重配置一键同步应用至课程下的所有班级。
- **实时校验**: 自动计算权重总和，确保总和为 100% 时才允许保存，防止配置错误。

### 3. 统计数据导出与管理 (Statistics)
- **多类型支持**: 除了考勤数据外，新增了 **测验**、**作业**、**综合成绩** 的导出支持。
- **历史记录管理**: 支持在下载中心直接删除已生成的历史导出记录。
- **下载功能**: 修复了下载链接的获取逻辑，支持直接调用默认浏览器下载导出文件。

### 4. 教师团队管理 (Teacher Team)
- **界面集成**: 新增教师团队管理入口，支持查看和管理课程团队成员（部分功能开发中）。

---

## 🛠 优化与改进 (Improvements)

- **UI/UX 升级**: 
  - 登录界面适配暗黑模式 (Dark Theme)。
  - 管理界面采用卡片式布局，操作更直观。
  - 优化了下拉菜单和按钮的交互动效。
- **打包部署**:
  - 优化了 macOS 打包脚本 (`build.sh` + `.spec`)，生成单一可执行文件。
  - **路径修复**: 修复了打包后 `data` 目录路径识别错误的问题，现在支持在可执行文件同级目录下读取配置文件。
- **性能优化**: 
  - 采用多线程 (Worker Threads) 处理耗时网络请求，避免界面卡顿。
  - 增加了加载状态提示，提升用户等待体验。

## 🐛 问题修复 (Bug Fixes)

- 修复了“导入课程”界面闪退的问题。
- 修复了删除统计报表时 ID 提取失败的问题。
- 修复了打包后无法读取 `signin_plans.json` 等配置文件的路径错误。
- 修正了部分 API 请求头缺失导致学习通返回 403/404 的问题。
