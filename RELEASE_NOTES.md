# 版本说明 (Release Notes) - V0.5.0

## 📅 更新日期: 2026-04-11

这是《学习通教学资源管理系统》具有里程碑意义的 **V0.5.0** 版本！🚀 标志着系统从基础辅助工具正式进化为全方位的教学资源管理平台。

## ✨ 重大更新与新增特性 (New & Core Features)

### 1. ☁️ 云盘管理系统 (Cloud Drive)
- **API 深度集成**: 实现了云盘功能的完整协议对接，支持查看和管理个人/课程云端资源（部分功能持续完善中）。

### 2. 📚 题库系统 2.0 (Question Bank Evolution)
- **分页与矩阵支持**: 题库列表支持高效的分页查询，并新增了对**矩阵 (Matrix)** 等复杂数学结构的渲染支持。
- **增强型 Markdown**: 支持 Markdown 代码块、粗体等多种格式转换，满足计算机及理工科题目的排版需求。

### 3. 📝 作业闭环管理 (Homework Management)
- **作业库视图**: 新增 `Homework Library` 视图，方便快速检索和复用历史作业任务。
- **在线创建流程**: 深度集成 `Homework Create` 功能，支持从题库直接选取内容生成新的作业。

### 4. 📐 科学符号与公式支持 (KaTeX)
- **全站 KaTeX 集成**: 题目预览、内容展示均已支持 **KaTeX 数学公式渲染**，提供印刷级的公式显示效果。

### 5. 🛠️ 构建与稳定性优化
- **全平台构建脚本更新**: 优化了 Windows, macOS, Linux 的 `.spec` 打包配置。
- **依赖库规范化**: 同步更新 `requirements.txt`，确保环境部署的一致性。

# 版本说明 (Release Notes) - V0.3.0

## 📅 更新日期: 2026-04-06

这是《学习通教学资源管理系统》的 **V0.3.0** 里程碑版本！🎉 本次更新引入了核心“活动”管理模块及全异步架构升级。

## ✨ 重大更新与新增特性 (New & Core Features)

### 1. 🚀 核心“活动管理”系统上线
- **批量签到计划**: 支持按照周次、单双周规律一键生成全学期签到预设。
- **逆序一键发布**: 完美解决学习通“先发后显”逻辑，支持将本地预设的签到任务逆序批量推送到服务器。
- **活动分组管理**: 实现对课程分组的同步、重命名、新增及彻底删除，完美适配教师端管理场景。

### 2. 🏗️ 全异步 Worker 架构重构
- **深度多线程化**: 所有网络请求及耗时操作（发布、同步、删除）均已迁移至后台 Worker 线程。
- **界面零掉帧**: 无论是在进行批量发布还是数据同步，UI 界面依然保持丝滑顺畅。
- **稳健的 API 层**: 统一了数据模型与网络层接口，大幅降低了接口访问失败的概率。

### 3. 🎨 UI/UX 与个性化增强 (HaoMD)
- **HaoMD 预览优化**: 优化了 KaTeX 公式渲染性能，支持更复杂的数学公式展示。
- **配置持久化**: 增强了“配置记忆”功能，自动记录并恢复上次选择的班级、分组及签过设置。
- **批量发布进度**: 新增实时任务队列监控，发布进度一目了然。

### 4. 🛠️ 稳定性与维护性
- **路径自平衡**: 优化了 `SIGNIN_DATA_FILE` 等本地存储路径，确保跨平台部署时的文件隔离。
- **错误捕获**: 完善了 API 返回值的深度校验。

# 版本说明 (Release Notes) - V0.1

## 📅 更新日期: 2026-01-24

## ✨ 新增功能 (New Features)

### 1. 班级管理系统 (Class Management)
- **批量导入学生**: 支持解析树维教务系统导出的 Excel 成绩登记册，将学生批量添加至指定班级。
- **班级维护**: 新增重命名班级、归档（删除）班级功能。
- **教师分配**: 支持将班级分配给教学团队中的特定教师。
- **新建班级**: 提供了新建班级的入口（API已集成）。

### 2. 成绩权重管理 (Grade Weights)
- **可视化配置**: 提供了直观的百分比滑块/输入框，用于调整各项成绩权重（如视频、测验、作业、考试等）。
- **一键同步**: 支持将当前权重配置一键同步应用至课程下的所有班级。
- **实时校验**: 自动计算权重总和，确保总和为 100% 时才允许保存，防止配置错误。

### 3. 统计数据导出与管理 (Statistics)
- **多类型支持**: 除了考勤数据外，新增了 **测验**、**作业**、**综合成绩** 的导出支持。
- **历史记录管理**: 支持在下载中心直接删除已生成的历史导出记录。
- **下载功能**: 修复了下载链接的获取逻辑，支持直接调用默认浏览器下载导出文件。

### 4. 教师团队管理 (Teacher Team)
- **界面集成**: 新增教师团队管理入口，支持查看和管理课程团队成员（部分功能开发中）。

## 🛠 优化与改进 (Improvements)

- **UI/UX 升级**: 
  - 登录界面适配暗黑模式 (Dark Theme)。
  - 管理界面采用卡片式布局，操作更直观。
  - 优化了下拉菜单和按钮的交互动效。

- **性能优化**: 
  - 采用多线程 (Worker Threads) 处理耗时网络请求，避免界面卡顿。
  - 增加了加载状态提示，提升用户等待体验。

## 🐛 问题修复 (Bug Fixes)

- 修复了“导入课程”界面闪退的问题。
- 修复了删除统计报表时 ID 提取失败的问题。
- 修复了打包后无法读取 `signin_plans.json` 等配置文件的路径错误。
- 修正了部分 API 请求头缺失导致学习通返回 403/404 的问题。
