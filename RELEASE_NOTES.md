# 版本说明 (Release Notes) - V0.5.6

## 📅 更新日期: 2026-04-29

### ✨ LaTeX 公式渲染引擎重构 (KaTeX Rendering)
- **本地化高质量渲染**: 引入了基于无头浏览器 (Headless WebEngine) 的本地 KaTeX 快照渲染机制。现在处理复杂数学公式（如矩阵、求和、积分等）时，将优先使用本地 KaTeX 引擎，不仅提升了渲染速度，还大幅提高了图片排版质量，有效解决网络受限时的公式丢失问题。
- **解析算法增强**: 深度扩展了 LaTeX 符号映射库，完美支持变量被 `[]` 包裹时的特殊上下标结构转换（如数组下标）。
- **图片排版自适应**: 优化了生成的公式图片的 HTML 标签属性，引入 `max-width: 100%` 和自适应宽度，确保超长公式在各端显示时不会破坏版面。

### 🛠️ 稳定性与构建升级
- **CI/CD 自动化冒烟测试**: 在 GitHub Actions 流水线中引入了环境依赖包（如 `xvfb`）以支持 Linux 端的无头化运行，并在正式打包前加入 **KaTeX 冒烟测试 (Smoke Test)**，彻底杜绝了因环境缺失导致公式引擎静默失效的风险。
- **体验修复**: 修复了题库目录树在根节点 (`root`) 刷新时逻辑不生效的 Bug。
